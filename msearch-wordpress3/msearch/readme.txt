== Description ==

Multi Search (aka msearch) is a plugin that uses the inherent search facilities of various systems, all in one plugin.
This is a fully customizable plugin that enables you to give your website visitors the option to search in the native
WordPress search system, Google, Bing and Yahoo. Especially for the three big search engines there is the ability to
search the entire web or whithin your website through their engine.

You can find more information at the plugin's homepage http://www.creativepulse.gr/en/appstore/msearch

== Installation ==

Use the regular installation procedure to install this plugin.

That is, go to Admin > Plugins > Add New > Upload.

Alternatively you can upload it into the `/wordpress/wp-content/plugins/` directory
and then activate it in the Admin control panel for plugins.
